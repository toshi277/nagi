#　サイトテーマ

静かなサウナ体験を利用者同士で共有・評価できるレビュー型コミュニティサイト

# テーマを選んだ理由

もともとサウナが好きで、リフレッシュや気分転換のために定期的に利用していますが、実際に訪れてみるまで「どの程度静かに過ごせる環境なのか」が分からず、想像していた体験と異なることがありました。
多くのサウナ情報サイトでは、設備や温度、立地などの情報は充実している一方で、利用者のマナーや施設全体の雰囲気、静かさといった体験の質に関する情報は十分に共有されていないと感じています。
同じように「落ち着いた環境で静かにサウナを楽しみたい」と考えている利用者にとって、事前に施設の雰囲気を把握できるサービスがあれば、より満足度の高いサウナ選びにつながると考えました。
そこで、静かさという視点に特化し、利用者同士が体験を共有・評価できるポートフォリオを制作してみようと考え、このテーマを選びました。

# ターゲットユーザー

静かな環境で落ち着いてサウナを楽しみたいと考えている人

初めて訪れるサウナ施設の雰囲気を事前に把握したい人

自身のサウナ体験を他の利用者と共有したい人

# 主な利用シーン

新しいサウナ施設を訪れる前に、施設の雰囲気や静かさを確認したい時

実際に利用したサウナの体験や静かさを記録・共有したい時

落ち着いて過ごせるサウナを探したいと感じた時
