静かなサウナ体験を共有・評価できるレビュー型コミュニティサイト

② テーマを選んだ理由（※3文以上・ですます調）

もともとサウナが好きで、リフレッシュや気分転換のために定期的に利用していますが、実際に行ってみるまで「どの程度静かに過ごせるのか」が分からず、期待と違う体験になることがありました。
多くのサウナ情報サイトでは、設備や温度、立地といった情報は充実している一方で、利用者のマナーや静かさといった体験の質に関する情報が十分に共有されていないと感じています。
同じように「落ち着いて静かにサウナを楽しみたい」と考えている利用者が、事前に雰囲気を把握できるサービスがあれば、満足度の高いサウナ選びにつながると考えました。
そこで、静かさという視点に特化して利用者同士が体験を共有できるポートフォリオを制作してみようと考え、このテーマを選びました。

③ ターゲットユーザー（2つ以上）

静かな環境でサウナを楽しみたいと考えている人

初めて訪れるサウナの雰囲気を事前に知りたい人

自分のサウナ体験を他の利用者と共有したい人

④ 主な利用シーン（2つ以上）

新しいサウナ施設を訪れる前に、雰囲気を確認したい時

実際に利用したサウナの体験や静かさを記録・共有したい時

落ち着いて過ごせるサウナを探したい時
